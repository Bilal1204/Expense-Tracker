
# Expense Tracker

Track you daily transactions in 3 categories namely Savings, Expenses, Investment




## Installation

Install it with
```bash
  cd client  
  npm i
  cd server 
  npm i
```
Run this project with
```bash
  cd client  
  npm start
  cd server 
  npm start
```
